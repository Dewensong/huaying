# 贡献指南

感谢你对智播坊的关注！欢迎任何形式的贡献。

## 如何贡献

### 报告 Bug
1. 在 Issues 中搜索是否已有相同问题
2. 提供清晰的描述、复现步骤、期望行为、截图（如有）
3. 标注环境信息（操作系统、浏览器、Node.js 版本）

### 提交功能建议
1. 说明功能的使用场景
2. 描述期望的效果
3. 如果有参考案例可以附上链接

### 提交代码
1. Fork 本仓库
2. 创建新分支：git checkout -b feature/your-feature
3. 提交更改：git commit -m "描述你的改动"
4. 推送分支：git push origin feature/your-feature
5. 创建 Pull Request

### 代码规范
- 使用 TypeScript 严格模式
- 前端组件使用 Vue 3 Composition API
- 提交信息遵循约定式提交（Conventional Commits）
- 确保 npm run build 无报错
